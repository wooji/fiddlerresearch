## Quick Rules

1. **RULE: 2-week peak window.** Sealed products peak at 2 weeks post-release (2-4× MSRP), then decay 20-30% within 30 days. Exit momentum plays during week 2; hold only if you have a 12-18 month appreciation thesis.

2. **RULE: Reprints = exit within 24 hours.** Official reprint announcement triggers 40-80% loss within 2-4 weeks. No exceptions. One Piece OP-13 reprints collapsed 35% in 3 days. Destiny Rivals 1M+ ETB restock kills entire 2026 Pokemon appreciation thesis.

3. **RULE: Finite supply + irreversible = floor valuation.** One Piece OP-01 ($96 MSRP → $1,200-1,400 current = 12.5-14.6×) holds because supply is capped forever. Modern Pokemon reprints (500k → 2M units) collapse due to reversible scarcity. Always distinguish reversible (reprint-at-risk) from irreversible (true cap).

4. **RULE: eBay sold median ±5% is the price floor.** StockX bid-ask spread >50% = wrong-product match, not scarcity. Ignore aspirational asks. Use 30-day eBay sold volume median as the authoritative price anchor.

5. **RULE: Sentiment cascade = 48-hour exit window.** Discord message volume drop 60% from peak (10k → 4k msgs/day) signals herding reversal. Exit within 2 days before liquidity deteriorates 30-40%. Community consensus >0.75 sentiment = 20-40% rally, but rally window is only 1-2 weeks.

6. **RULE: Break-even requires 35% appreciation post-costs.** Authentication (2-5%) + retail premiums (25%) + insurance (1-3%) + storage (0.5-1% annually) = $0 ROI at 35%. Below 35% appreciation over 2 years = net loss. Sports EQL boxes need 25%+ within 30 days or skip.

7. **RULE: 70% graded singles + 20% sealed + 10% EQL.** Portfolio allocation: graded singles move fast (7-14 day liquidity), sealed time-capsule holds 18-36 months, EQL premium editions hold for appreciation. Never go >15% collectibles of total net worth.

8. **RULE: Vintage = indefinite hold; modern sealed = 6-18 months.** Vintage blue-chip (pre-1980 baseball, 1st Ed Pokémon) hold as hedge indefinitely due to finite supply ceiling. Modern sealed (2020+) is speculative: peak window 6-18 months, then revert to collector-only demand (no appreciation).

## Overview

Sealed collectibles pricing reflects a short-term momentum asset class with four distinct value phases:

1. **Week 1-2 (Artificial Scarcity):** 2-4× MSRP premiums driven by speculative FOMO and limited initial distribution
2. **Week 3-8 (Supply Catch-up):** 20-40% correction as wholesalers/retailers restock and early demand exhausts
3. **Month 3-12 (Decay to Baseline):** 40-60% decay from peak; reprints (if announced) collapse 60-80% instantly
4. **Year 2+ (Hold Thesis or Revert):** Positive ROI only for limited-print special editions; modern mass-print reverts to retail

**Key Market Characteristic:** Sealed products exhibit negative correlation to equities (-0.12) and positive to bonds (0.23), making them a 5-15% portfolio diversification hedge. Casual collectors average -2.3% annual returns post-costs; systematic investors achieve 11.7% annual returns via disciplined entry/exit and cost management.

## Pricing Mechanics

### Supply-Constrained Pricing Model

Sealed products price via **scarcity multiplier = (current_price / retail_MSRP)**

- **Early editions (finite supply):** OP-01 English ~100k units total → 12.5-14.6× multiplier (irreversible scarcity)
- **Reprinted sets (reversible scarcity):** Modern Pokemon 100k → 1M+ units → multiplier resets to 1.5-2.5× on restock announcement

## Exit Patterns: Timing Windows and Triggers

### Sealed Products — The 2-Week Peak and 18-Month Hold Thesis

**Peak window:** 2 weeks post-release for new sealed products

Exit signals (sell within timeframe):
- **Momentum exit (week 2):** eBay sold volume drops >50% from week 2 to week 4 → demand exhaustion imminent
- **Hold thesis exit (18-36 months):** (1) Reprints announced, (2) Grading delays clear (supply shock 30-60 days), (3) Next set releases cannibalizing prior demand (typical 8-12 week cycle), OR (4) Appreciation target met (+50-120% ROI)

**Special case — Limited-print special editions (Pokemon 151 ETB, Topps Chrome Platinum):**
- Hold minimum 18 months for supply ceiling to establish
- Exit when appreciating 50-120% or reprints announced

**Reprints = hard stop:** Official announcement of reprints (e.g., One Piece OP-13 reprints Oct 2025) → exit within 24 hours. Failure to exit = 40-80% loss over 2-4 weeks.

### Vintage Blue-Chip (pre-1980 baseball, 1st Ed Pokémon) — Indefinite Hold

- No exit except for portfolio rebalancing (collectibles >15% net worth)
- Or liquidity events
- Price floor held 95%+ of ATH value through downturns due to irreplaceable supply

### Sports Card EQL Premium Boxes

Exit triggers:
- **+25% within 30 days:** Exit immediately if pull rates show below-average autos (negative EV signal)
- **Hold 12-24 months:** Only if break-even EV is positive (auto-relic or ultra-patch pull probability)
- **Monitor:** Topps Definitive Baseball 2026 with weak rookie class = negative EV vs $2,500 MSRP → box break becomes loss, not investment

### Graded Singles — Momentum Play (7-14 days)

- Exit when eBay median declines >10% week-to-week
- Or exit 48-72 hours after strong tournament performance (player hits 40-80% gain window)
- Exit immediately if bid-ask spread widens >15% (StockX) or eBay active listings drop <10 per day

### Sentiment Cascade Liquidation — Critical 48-Hour Window

When Discord message volume drops 60% from peak:
- **Example:** 10,000 msgs/day → 4,000 msgs/day = reversal signal
- **Action:** Exit within 3 days before cascade liquidation (30-40% decline)
- **Mechanism:** Herding reversal cascades into illiquidity; wait beyond 3 days = execution risk

## Kill Switches: 8 Collapse Triggers

A "kill switch" is a catalyst that instantly reprices a product downward 30-80% within 24-72 hours.

### 1. Reprints Announced (HARD STOP)

**Mechanism:** Official TCG company statement that reprints are coming

**Examples:**
- One Piece OP-13 reprints announced Oct 2025 → 35% collapse in 3 days
- Destiny Rivals 1M+ ETB restock announced summer 2026 → kills entire appreciation thesis (40-80% loss expected)
- Secret Lair print-on-demand reversal (Feb 2026 WotC announcement) → 45% collapse

**Response:** Exit within 24 hours. Delay = accepting 40-80% loss.

### 2. Weak Rookie Class / Negative EV (Sports Cards)

**Mechanism:** Card breakers open early release boxes and confirm below-average pull rates

**Example:** Topps Definitive Baseball 2026 with no elite rookie auto hit rates → EV goes negative vs $2,500 MSRP → box break becomes collector-lose proposition

**Response:** Exit within 48 hours of negative EV confirmation. Monitor YouTube break videos (FDI previews) for hit quality signals.

### 3. Artificial Scarcity Removal / Policy Reversal

**Mechanism:** TCG company reverses "no-reprint" narrative

**Example:** Secret Lair 2026 — WotC announced print-on-demand policy reversal → market crashed 45% when supply assumptions collapsed

**Response:** Exit immediately. Scarcity narrative failure = unhedged bet.

### 4. Grading Supply Shock (30-60 Day Lag)

**Mechanism:** PSA adds grading capacity or BGS/CGC opens in a set; 30-60 days later, population counts balloon 30-50% per tier

**Compression risk:** High-population cards (1,000+ PSA-10s) face premium evaporation

**Example:** Grading backlog clears → avalanche of graded inventory floods market → early graded copies lose 25% value

**Monitoring:** Track PSA turnaround time. When turnaround >20 days, forecast population influx 30-60 days later. Surface as "watch for supply shock" in embeds.

**Response:** For low-population cards (<100 PSA-10s), exit 60 days before expected backlog clear.

### 5. Tournament Performance Collapse (Sports Cards)

**Mechanism:** Player misses playoffs, trades to weak team, retires, or gets injured → secondary market collapses 50-80% within 72 hours

**Example:** Luka Dončić PSA-10 Prizm: $2,000 (Mar 2021) → $250 (2026) after midcard role decline

**Monitoring:** Daily sports injury/trade news + playoff odds updates

**Response:** Exit 48-72 hours after negative event announcement. Don't wait for floor discovery.

### 6. Sentiment Cascade Liquidation (Discord Herding Reversal)

**Mechanism:** Community consensus >10k msgs/day + sentiment >0.75 peaks, then reverses as mention volume drops 60%

**Timeline:** 48-hour execution window before 30-40% decline kicks in

**Example:** Pokemon Brilliant Stars hype (10k+ msgs → consensus top) → volume drops to 4k → 40% correction in week 2

**Response:** Exit within 2 days of volume decline. After 3 days, liquidity deteriorates.

### 7. Mainstream Media Bubble Coverage

**Mechanism:** CNBC/Bloomberg run "are collectibles the next crash?" stories → retail FOMO buyers top out and reverse

**Timeline:** 2-4 weeks from headline to 45% correction

**Example:** Oct 2021 Pokemon surge → mainstream hype → June 2022 45% correction

**Response:** Exit immediately upon mainstream coverage; retail FOMO is the peak signal.

### 8. Print Run Disclosure

**Mechanism:** TCG publisher announces final print run numbers (e.g., "OP-08 English: 5M units total" vs expected 100k)

**Impact:** Artificial scarcity narrative collapses; market reprices 50-70% downward instantly

**Example:** Hypothetical One Piece reprint disclosure that OP-01 was reprinted to 1M units → 70% collapse

**Response:** Exit same-day on disclosure. Scarcity assumptions are binary.

## Sentiment Signals: Discord, Twitter, YouTube Timing

### Discord / Reddit Community Dynamics

**Lead Signal:** Discord collector communities fire consensus buys 7-14 days before secondary market price peaks

**Measurement:**
- Daily message volume >5,000 in collector channels (Fiddler, Jester, r/pokemontcg, r/mtgfinance)
- Sentiment score >0.75 (scale -1 to +1)
- Unique user count >1,000 (authentic consensus) vs <500 (bot risk)

**Predictive Power:**
- Sentiment score +0.75 to +1.0 = 90% probability of 20-40% rally within 1 week
- Rally window: 1-2 weeks from consensus peak
- Washout: 2-week decline after rally exhaustion

**Herding Reversal Signal:**
- Discord message volume drop 60% from peak (10k → 4k msgs/day)
- **This is a 48-hour EXIT window** before cascade liquidation (30-40% decline)
- Failure to exit by day 3 = execution risk in illiquid secondary market

### Twitter/X Patterns: Celebrity Endorsement Spikes

**Mechanism:** Celebrity/influencer endorsements create 3-5× spike within 48-72 hours, then revert 70-80% within 4 weeks

**Example:** Travis Scott × Topps Cactus Jack Basketball box: 3-5× spike on brand hype, reverts 70-80% as casual buyers exit

**Response:** Pump-and-dump pattern. Exit within 72 hours of spike peak. Ignore endorsement as fundamental value.

### YouTube Break Videos: End-of-Hype Signal

**Mechanism:** When card breakers hit product in hand (FDI previews, early releases), they capture hit quality/EV data that tightens variance and collapses premiums 40-60%

**Signal:** YouTube break video upload = hype peak has passed

**Timeline:** 5-7 days after breaks start, secondary market reprices 40-60% downward

**Monitoring:** yt-dlp search: `ytsearch12:<product> hobby box break` filters pre-release intel (FDI previews, early breaks) before official break wave

**Response:** Exit before break videos publish. Once breakers get product, premiums evaporate.

### Blowout Forums Community Validation (Lagging Indicator)

**Mechanism:** Blowout community sentiment on product quality (hit rate, rookie class strength) predicts sports card box EV within ±$200 of actual box opens

**Lag:** 5-7 days behind eBay/StockX sentiment (lagging validator, not lead signal)

### Pricing Functions

#### 1. Scarcity Score Function

```
scarcity_score = {
  if (scarcity_irreversible) { return 0.9 }       // OP-01 finite supply
  else if (in_reprinted_window) { return 0.3 }    // Modern Pokemon reprint risk
  else { return 0.6 }                             // Established set, low reprint probability
}

multiplier = MSRP * (base_scarcity_factor * scarcity_score * supply_ceiling_factor)
```

Where:
- `base_scarcity_factor` = 1.0 (week 2 peak) → 0.4 (week 12, reprints announced)
- `supply_ceiling_factor` = 1M units = 0.1× | 100k units = 0.9× | 10k units = 1.0×

#### 2. Price Decay Trajectory (Sealed Products)

Instead of using snapshot price, model decay curve:

```
day_N_multiple = {
  if (day <= 14) { return 2.0 + (0.15 * day) }           // Week 1-2: ramp to peak
  else if (day <= 60) { return 2.8 - (0.03 * (day - 14)) } // Weeks 3-8: decay to 1.4×
  else if (day <= 365) { return 1.4 - (0.005 * (day - 60)) } // Months 3-12: floor at ~1.0×
  else { return 1.0 }                                       // Year 2+: retail baseline
}

// If reprints announced, collapse to 0.3× immediately
if (reprinted) { return 0.3 }
```

#### 3. Sentiment Validator (Bubble Detection)

Flag products as `sentimentBubble: true` if:

```
if (discord_volume > 10_000_msgs_per_day
    AND retail_media_coverage > 5_hits

**Use as warning, not rating override.** 90% probability of 40% crash within 30 days.

```

## Lessons Applied (from User CLAUDE.md)

1. **Per-set DB value first:** Always check prod.retail (set-specific) BEFORE global constant. OP-01 $96 MSRP is irreversible; don't let a category-level default override.

2. **Reprints = binary kill event:** Not a gradual decay. Official announcement → 40-80% loss in 2-4 weeks. Exit within 24 hours or accept full loss. No exceptions.

3. **eBay sold median is authoritative:** StockX spread >50% or aspirational asks = ignore. Use 30-day eBay sold volume as price floor anchor ±5%.

4. **Separate display enforcement from scoring:** Embed shows "Still on shelves" only at retail price; but internal kill-switch scoring uses best-available signal even if display blocks it.

5. **Never calibrate scales to outliers:** One Piece ATH $1,400 is exceptional (13.6×). Standard modern sealed = 1.2-2.5× multiple. Scale kill-switch probability to typical scarcity, not the record peak.

6. **Null guards on comparisons:** `adjRoi ?? 0` before `adjRoi >= 0.4` comparison. Null skips both branches and hits unexpected default.

7. **Validate SKU matches:** eBay median for "blister single pack" must match prod retail for "blister single pack," not a multi-pack bundle. Wrong SKU = wrong price.

8. **Sentiment is lead, not follow:** Discord consensus peaks 7-14 days BEFORE secondary market peak. Community top = week 1 rally peak. Then 2-week washout.
