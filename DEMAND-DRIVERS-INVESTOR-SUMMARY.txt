EXECUTIVE SUMMARY — 4 FACTORS THAT DETERMINE SEALED HOLD
================================================================================

1. IP TIER (50% of sealed multiple)
   ✓ S-tier (Pikachu, Charizard, Eevee) → 2.5-3.5× floor
   ✗ C-tier (Paradox, Pecharunt mythical) → 1.0-1.5× peak, crashes to <1.0×

2. RARITY TIER (25% of sealed multiple)
   ✓ Ex-rarity → +0.5-1.0× vs non-ex
   ✓ Mega ex → +0.3× additional premium
   ✗ V/Vmax only → weak hold, 1.2-1.5× max

3. MECHANIC STRENGTH (15% of sealed multiple)
   ✓ Core mechanics (Tera) → sustain 2-4× for 12+ months
   ✗ Gimmick mechanics (Paradox, Mythical) → spike 1.3-1.5× then crash in 3 months

4. SUPPLY SCARCITY (10% of sealed multiple)
   ✓ Tight supply (<1K units) + strong IP → 2.5-3.5× with minimal decay
   ✗ Massive print (10M+ units) + strong IP → compressed to 1.5-2.5×
   ✗ Tight supply + weak IP → brief spike then crash to retail

INVESTMENT DECISION MATRIX
================================================================================

BUY AGGRESSIVELY (DBLGREEN):
  ✓ S-tier IP (Pikachu, Charizard, Eevee, Starters)
  ✓ Ex rarity tier (Mega ex or standard ex)
  ✓ Tight supply (<1K units at major retailers)
  ✓ Core mechanic (Tera, not Paradox/Mythical)

  Examples: Ascended Heroes, Pitch Black, Destined Rivals presale
  Target: 2.5-3.5× within 6 months | 2.0-2.5× floor at 12+ months
  Exit: Sell half at +100% (6 months), hold half long-term

HOLD (GREEN):
  ✓ S-tier IP + ex rarity + normal supply
  ✓ A-tier IP + ex rarity (Lugia, Zapdos)
  ✓ Core mechanic with proven hold (Tera)

  Examples: Destined Rivals, Surging Sparks, Obsidian Flames
  Target: 2.0-2.5× floor for 12+ months | Slow bleed after 18 months
  Exit: Hold long-term, sell only if life happens

LIGHT BUY (ORANGE):
  △ A-tier IP but massive print (Prismatic Evolutions, Eevee)
  △ B-tier IP + normal supply (Journey Together)
  △ Generic legendary sets with weak chase

  Examples: Prismatic Evolutions, Phantasmal Flames presale only
  Target: 1.5-2.0× | Expect floor erosion by month 12
  Exit: Sell at +50-100%, don't hold long

AVOID (RED):
  ✗ Gimmick mechanics (Paradox, Pecharunt mythical)
  ✗ Non-ex rarity (V/Vmax only sets)
  ✗ Weak IP + normal supply
  ✗ Presale at inflated ask on hype

  Examples: Chaos Rising day-2 purchases, Pecharunt day-1, Journey Together
  Why: Crash pattern confirmed — spike to 1.3-1.5×, floor to <1.0× in 3 months
  If you already hold: Dump by month 2; losses will exceed holding

QUICK DECISION CHECKLIST
================================================================================

Before buying ANY Pokemon sealed set, ask:

1. IP TIER: Is it S-tier (Pikachu/Charizard/Eevee/Starters)?
   YES → continue | NO → rating drops to ORANGE or RED

2. RARITY TIER: Does it have ex rarity?
   YES → ex adds +0.5-1.0× | NO → floor capped at 1.5×

3. MECHANIC: Is it a core mechanic (Tera) or gimmick (Paradox/Mythical)?
   CORE → 12+ month hold potential | GIMMICK → 3-month crash risk

4. SUPPLY: Is stock <1K or is it massive print (10M+)?
   TIGHT + S-tier IP → 2.5-3.5× floor
   MASSIVE + S-tier IP → 1.5-2.5× (compressed)
   TIGHT + weak IP → brief spike then crash

5. RELEASE STATE: Is this presale or post-release?
   PRESALE + S-tier IP + ex + tight supply → BUY
   POST-RELEASE month 2+ + gimmick mechanic → AVOID

DECISION TREE:
  S-tier IP + ex + tight supply + core mechanic → DBLGREEN (buy aggressively)
  S-tier IP + ex + normal supply + core mechanic → GREEN (buy, hold long)
  A-tier IP + ex + core mechanic → ORANGE (wait for +30% or pass)
  Gimmick mechanic → RED (avoid entirely, crash to <1.0× in 3 months)
